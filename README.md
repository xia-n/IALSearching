1. Setup

Place searcher.exe in a folder with libssl-3-x64.dll and libcrypto-3-x64.dll. Run the EXE to begin.

3. Security & Access
On launch, the engine generates a unique hardware ID based on your CPU.

Prompt: [ID] XXXXXXXX... [AUTH] Verifying...

Action: If you see GRANTED, you’re in. If it says FAILED, send your ID to the admin to white-list your device. 
Via this form:https://docs.google.com/forms/d/e/1FAIpQLSdNNhdmO4xQFXKiKNPPrznCYS4jkonwYP-s7S76UjiHS4919g/viewform?usp=header
4. How to Search
Select Subject: Press 1, then choose the index number of the atlas you want to load.

Query: Type any keywords or phrases (e.g., “enthalpy change”).

Results: The engine shows the top 8 matches with a confidence percentage and a text snippet.

Navigation: Type b to switch subjects or 2 to exit the program.

5. Troubleshooting
OFFLINE: No internet. You can still use already downloaded files.

Signature Mismatch: Cloud files are out of sync or tampered with. Notify admin.

No atlas files: Your /subjects folder is empty. Connect to Wi-Fi and restart to sync.
